 # Inicializacion de JasperReport:
 * descargar e instalar JASPERREPORT
 * instalar los plugin para el funcionamiento del reporte del programa de: https://drive.google.com/drive/u/0/folders/17CbisR1BdyGMYVtXg7jJ4ZheMqCyt6FJ
 * crear tu reporte utilizando report wizard
 * crear tu query que quieres hacer tu reporte
 * en el report wizard pones el nombre y clave de tu base de datos ademas del gestor de base de datos que estas utilizando
 * pegas tu query en el report wizard y lo testeas para ver si conecta a tu base de datos
 * en el reporte crear dos parametros para el conteo de clientes y la suma de los costos de la reserva de habitacion
 * conectar a tu base de datos
 * en la capa de persistencia crear tu funcion con los parametros de las dos fechas
 * crear tu variable donde vas a poner tu query 
 * crear dos funciones mas con sus parametros de dos fechas para hacer uso para el hashMap
 * en la capa de aplicacion crear tu funcion con los parametros de las dos fechas
 * crear el codigo para el JasperReport. Para mas informacion: https://www.youtube.com/watch?v=AbawnPImW_g
 * crear el hashMap para el parametro del reporte. Para mas informacion: https://www.w3schools.com/java/java_hashmap.asp
 * crear dos variables para poder llamar a las funciones que seran utilizados como parametros del reporte
 * finalmente en la capa de presentacion creas dos variables para dar uso a las fechas
 * llamas a tu funcion de la capa de aplicacion
